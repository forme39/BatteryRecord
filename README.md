
**BatteryRecord** 是一款 Android 深度电池分析工具，通过可视化图表将电量消耗与 App 使用记录精准匹配。

### 🌟 核心功能

* **交互式图表**：折线图展示电量趋势，统计已使用时间、亮屏时间。
* **硬件监控**：实时追踪 **功率(W)**、**温度(°C)** 和 **电压(V)**。
* **续航分析**：自动计算平均功耗，并预测理论总续航时间。
* **历史回溯**：记录详细的充放电轨迹和能耗统计数据。

### 🛠️ 技术栈

* **图表**: MPAndroidChart
* **构建**: Java && Gradle

### 📸 界面预览

应用主界面包含：**电量/应用复合曲线图**、**实时参数卡片**以及**续航统计看板**。
